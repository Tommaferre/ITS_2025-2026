Introduzione

Benvenuto alla guida per creare la tua prima pagina HTML. In questa pagina imparerai le basi per strutturare un documento HTML, aggiungere stili con CSS e rendere interattiva la tua pagina con JavaScript.

HTML - La struttura della pagina

L'HTML è il linguaggio di markup utilizzato per creare la struttura delle pagine web. Utilizza tag per definire elementi come titoli, paragrafi, immagini, link e molto altro.

<!DOCTYPE html>
<html lang=\"it\">
  <head>
    <meta charset=\"UTF-8\">
    <title>La mia prima pagina HTML</title>
  </head>
  <body>
    <h1>Ciao mondo!</h1>
    <p>Benvenuto nel mondo del web.</p>
  </body>
</html>

CSS - Lo stile della pagina

Il CSS permette di definire lo stile dei tuoi elementi HTML, come colori, layout, spaziatura e molto altro. Nel nostro esempio, abbiamo scelto un tema scuro per un aspetto formale ed elegante.

body {
  background-color: #121212;
  color: #e0e0e0;
  font-family: 'Segoe UI', sans-serif;
}

JavaScript - L'interattività

JavaScript è il linguaggio di programmazione che aggiunge interattività alle pagine web. Puoi usarlo per creare animazioni, gestire eventi e modificare dinamicamente il contenuto della pagina.

document.addEventListener('DOMContentLoaded', function() {
  console.log('La pagina è pronta!');
});

Presentazione Interattiva

Clicca sui pulsanti per navigare tra i livelli di complessità del codice HTML.

Livello 1 - Base

<!DOCTYPE html>
<html>
  <head>
    <title>Pagina Base</title>
  </head>
  <body>
    <h1>Ciao Mondo!</h1>
  </body>
</html>

Livello 2 - Con CSS

<!DOCTYPE html>
<html>
  <head>
    <title>Pagina con Stile</title>
    <style>
      body { background-color: #222; color: white; }
    </style>
  </head>
  <body>
    <h1>Pagina con Stile!</h1>
  </body>
</html>

Livello 3 - Con Interattività

<!DOCTYPE html>
<html>
  <head>
    <title>Pagina Interattiva</title>
    <script>
      function saluta() {
        alert('Ciao dal JavaScript!');
      }
    </script>
  </head>
  <body>
    <h1>Pagina Interattiva!</h1>
    <button onclick="saluta()">Cliccami</button>
  </body>
</html>
